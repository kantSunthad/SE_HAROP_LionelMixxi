อาจารย์ครับเนื่องจาก พอโอนถ่ายไฟล์งาน Xcode จาก mac มา windows แล้วชื่อ folder มันทับซ้อนกัน ผมเลยแก้ปัญหาโดยการให้ ไฟล์ zip ไปแล้วให้อาจารย์ แตกไฟล์ รันโค้ดเอาครับ
1) แตกไฟล์ mapApp.zip
2)เข้า mapApp2(เพราะชื่อซ้ำใน folder มันจะเติมเลข 2 ให้)
3)mapApp.xcodeproj(ขั้นตอนหลังจากนี้ใส่ไว้ให้ใน manual ครับ)

ที่จะส่งจริงๆมีไฟล์ตามนี้ครับ 
-mapApp.zip -HOROP_MANUAL.docx -FinalReport.pdf

***คือไฟล์ Code ทั้งหมด อยู่ใน mapApp.zip ครับ ไฟล์ที่เหลืออาจารย์ไม่ต้องสนใจก็ได้ครับ มัน ชื่อซ้ำจนอัพไม่ได้*****
